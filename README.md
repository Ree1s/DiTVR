# Supplementary Materials for CVPR Workshop NTIRE Paper Submission (ID 110)

## DiTVR: Zero-Shot Diffusion Transformer for Video Restoration

This repository contains the supplementary materials for an anonymous submission to the CVPR Workshop NTIRE track.

### Contents

- **Teaser Video**: Demonstrates the proposed method's capabilities in video restoration tasks, including super-resolution, denoising, deblurring, and colorization.
- **Video Comparisons**: Side-by-side comparisons on the DAVIS and SPMC datasets for both super-resolution and denoising tasks.
- **Paper Abstract**: A summary of our research and findings.

### How to View

Open `index.html` locally in your browser, or publish this folder with GitHub Pages.

### GitHub Pages Deployment

This project is already structured as a static site and is suitable for deployment at a URL like `https://xxx.github.io/yyy/`.

Recommended anonymous deployment workflow:

1. Create a new anonymous GitHub account or organization for the review period.
2. Create a new public repository, for example `anonymous-project-page`.
3. Upload the contents of this folder so that `index.html` is in the repository root.
4. Keep `.nojekyll` in the root so GitHub Pages serves the site as plain static files.
5. In GitHub, open `Settings` -> `Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select the `main` branch and the `/ (root)` folder, then save.
8. Wait for GitHub Pages to publish the site. The final URL will look like `https://<account>.github.io/<repo>/`.

### Anonymous Review Checklist

- Keep the page title, metadata, and visible text anonymous.
- Do not add author names, affiliations, personal domains, or social links.
- Avoid pushing from a personal GitHub account if the venue is double blind.
- Check repository history and description before sharing the link.
